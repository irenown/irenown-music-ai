javascript// Simplified Backend Flow

1. UPLOAD ENDPOINT
   - Receive audio file
   - Store in S3/R2 with unique ID
   - Queue for processing

2. VOCAL PROCESSING JOB
   - Send to Adobe Podcast API
   - Receive enhanced vocal
   - Analyze with Essentia.js (detect key, tempo, duration)
   - Store processed vocal

3. MUSIC GENERATION JOB
   - Send parameters to Mubert/Loudly API:
     * Genre selection
     * Detected key
     * Detected tempo
     * Duration
   - Receive instrumental track

4. MIXING JOB
   - Use FFmpeg to:
     * Align tempo/pitch if needed
     * Mix vocal and instrumental
     * Apply basic EQ/compression
     * Normalize levels
   - Optional: Send to LANDR for final master

5. DELIVERY
   - Store final track
   - Notify user
   - Provide download link
